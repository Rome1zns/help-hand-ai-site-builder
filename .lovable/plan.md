

# Создание демо-лендингов для карточек галереи

## Обзор
Для каждой из 6 карточек в секции "Примеры сайтов" будет создана отдельная страница-лендинг внутри проекта. При клике на карточку пользователь перейдёт на соответствующий демо-сайт и увидит живой, красивый лендинг.

---

## Новые страницы (6 штук)

### 1. `/demo/coffee` — Кофейня "Арома"
- Тёплая палитра (amber/brown)
- Hero с фото кофейни и слоганом
- Секция меню (кофе, десерты, завтраки)
- Секция "О нас" с атмосферными фото
- Форма бронирования столика (визуальная, без бэкенда)
- Контакты и карта

### 2. `/demo/construction` — СтройГрупп
- Строгая тёмная палитра (slate/gray)
- Hero с фото строительства
- Секция услуг (строительство, ремонт, проектирование)
- Портфолио объектов (сетка с фото)
- Секция "Почему мы" с цифрами (20+ лет, 500+ объектов)
- Контакты

### 3. `/demo/techstore` — TechStore
- Современная палитра (blue/dark)
- Hero с фото техники
- Каталог товаров (карточки: ноутбуки, смартфоны, аксессуары)
- Секция акций / скидок
- Преимущества (доставка, гарантия, поддержка)
- Футер

### 4. `/demo/fitness` — Фитнес-клуб
- Энергичная палитра (green/dark)
- Hero с фото тренажёрного зала
- Расписание тренировок (таблица/сетка)
- Тарифы абонементов (карточки)
- Тренеры (аватары + описание)
- CTA "Записаться"

### 5. `/demo/travel` — Блог путешественника
- Тёплая палитра (orange/amber)
- Hero с фото дороги
- Последние статьи (карточки с фото)
- Популярные маршруты
- Галерея фотографий
- Подписка на рассылку

### 6. `/demo/design` — Студия дизайна
- Креативная палитра (purple/violet)
- Hero с фото рабочего пространства
- Портфолио кейсов (сетка)
- Услуги (логотипы, брендинг, веб-дизайн)
- Отзывы клиентов
- Контактная форма

## Общие элементы демо-страниц
- Каждая страница — самостоятельный красивый лендинг
- Кнопка "Назад на главную" в навбаре каждого демо-сайта
- Используются Unsplash-фото для визуала
- Адаптивный дизайн (мобильная версия)
- Framer Motion анимации для плавного появления секций
- Все данные статические (без бэкенда)

---

## Технические детали

### Структура файлов
```text
src/pages/demos/
  CoffeeDemo.tsx
  ConstructionDemo.tsx
  TechStoreDemo.tsx
  FitnessDemo.tsx
  TravelDemo.tsx
  DesignDemo.tsx
```

### Навигация
- Карточки в Gallery станут ссылками (`<Link to={p.link}>`)
- Каждая демо-страница содержит кнопку "Назад" (`<Link to="/#gallery">`)

### Стилизация
- Каждый лендинг использует свою цветовую палитру через Tailwind
- Используются уже существующие компоненты UI (Button, Card) где уместно
- Glass-эффекты и анимации из текущего проекта

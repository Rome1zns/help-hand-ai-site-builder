

# Исправление отсутствующего изображения в карточке "Студия дизайна"

## Проблема
В секции Gallery карточка "Студия дизайна" отображается без фотографии — текущий URL с Unsplash не загружается.

## Решение
Заменить нерабочий URL изображения на другой, рабочий URL с Unsplash, подходящий по тематике (рабочее место дизайнера / дизайн-студия).

## Технический план

**Файл: `src/components/landing/Gallery.tsx`**
- Строка с `image` у проекта "Студия дизайна" — заменить URL `photo-1558618666-fcd25c85f82e` на рабочий, например `photo-1561070791-2526d30994b5` (дизайнерское рабочее пространство).



# Help Hand — Landing Page

## Overview
A polished, dark-themed landing page for "Help Hand" — an AI no-code website builder. Interface fully in Russian. Dark background (#0a0a1a) with purple accent (#7c3aed), glassmorphism effects, and smooth animations.

## Design System Updates
- Dark theme as default with custom colors: background #0a0a1a, accent purple #7c3aed
- Inter font, glassmorphism cards (backdrop-blur, semi-transparent borders)
- Framer Motion for scroll-triggered and interactive animations

## Page Sections

### 1. Navbar
- "Help Hand" logo (left), navigation links (center), "Войти" and "Начать бесплатно" buttons (right)
- Sticky header with blur background on scroll
- Mobile hamburger menu

### 2. Hero Section
- Bold headline: "Создайте сайт за 30 секунд с помощью ИИ"
- Subtitle explaining the value proposition
- Large glowing prompt input field with send button
- Clickable example prompts underneath (e.g., "Лендинг для кофейни", "Портфолио дизайнера", "Интернет-магазин")

### 3. Animated Preview
- Mock browser window showing a website being "generated" in real-time
- Animated typing effect in the prompt area, then sections appearing one by one
- Glassmorphism frame around the preview

### 4. Gallery of Examples
- Grid of 6 project cards showing websites built on the platform
- Each card with screenshot, title, and category tag
- Hover effects with scale and glow

### 5. Pricing (Тарифы)
- Three pricing tiers: Бесплатный / Про / Команда
- Feature comparison list for each tier
- "Популярный" badge on Pro plan
- Glassmorphism cards with purple highlight on featured plan

### 6. Footer
- Logo, navigation links, social media icons
- Copyright text
- Minimal and clean design

## Technical Notes
- Add `framer-motion` package for animations
- Mobile-first responsive design throughout

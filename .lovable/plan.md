

# Добавить кнопку "Начать" в навбар

## Что будет сделано
В навигационную панель добавится заметная кнопка "Начать", которая будет вести пользователя в редактор (`/editor`). Кнопка будет видна и на десктопе, и на мобильном — чтобы пользователь сразу понимал, куда нажать.

## Технический план

**Файл: `src/components/landing/Navbar.tsx`**
- Импортировать `Button` из `@/components/ui/button` и `useNavigate` из `react-router-dom`
- На десктопе: добавить кнопку "Начать" справа от ссылок навигации, стилизованную как `btn-premium` (градиент, свечение)
- На мобильном: добавить кнопку "Начать" внутри мобильного меню, а также показать её рядом с бургер-иконкой для быстрого доступа
- Кнопка ведёт на `/editor` через `useNavigate`

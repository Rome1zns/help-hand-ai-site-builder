

# Улучшение мобильного дизайна лендинга Help Hand

## Обзор
Полная адаптация всех секций лендинга для красивого отображения на мобильных устройствах. Текущий дизайн уже частично адаптивен, но многие элементы выглядят слишком крупно или неудобно на маленьких экранах.

---

## Изменения по файлам

### 1. `src/components/landing/Navbar.tsx`
- Уменьшить вертикальный padding навбара на мобильном (`py-3` вместо `py-5`)
- Сделать логотип чуть меньше на мобильном

### 2. `src/components/landing/HeroSection.tsx`
- Уменьшить заголовок на мобильном: `text-3xl` вместо `text-5xl`
- Уменьшить подзаголовок: `text-base` на мобильном
- Уменьшить отступ badge сверху: `mb-6` вместо `mb-10`
- Примеры промптов: сделать горизонтальный скролл вместо `flex-wrap` на мобильном
- Статистика: уменьшить gap и размер цифр (`text-xl` на мобильном), расположить в ряд с меньшими отступами
- Уменьшить радиальные градиенты на мобильном (меньше размер)
- Уменьшить `mt` и `mb` отступы между блоками

### 3. `src/components/landing/AnimatedPreview.tsx`
- Уменьшить padding секции на мобильном: `py-16` вместо `py-32`
- Уменьшить заголовок: `text-2xl` на мобильном
- В сетке карточек внутри превью: `grid-cols-1` вместо `grid-cols-3` на мобильном (показать только 2 карточки)
- Уменьшить внутренние padding

### 4. `src/components/landing/Features.tsx`

- Карточки: уменьшить padding `p-5` на мобильном

### 5. `src/components/landing/Gallery.tsx`

- Заголовок: `text-2xl md:text-5xl`

### 6. `src/components/landing/Testimonials.tsx`
- Padding секции: `py-16 md:py-32`

## Технические детали

- Все изменения через адаптивные классы Tailwind (`text-2xl md:text-5xl`, `py-16 md:py-32` и т.д.)
- Никаких новых зависимостей
- Никаких изменений в логике или данных
- Только визуальные классы для мобильных размеров



# Добавить большую кнопку "Начать создавать" в центр Hero-секции

## Что будет сделано
Под полем ввода промпта добавится большая, заметная кнопка "Начать создавать", которая сразу бросается в глаза пользователю. Кнопка будет вести в редактор (`/editor`).

## Технический план

**Файл: `src/components/landing/HeroSection.tsx`**

- Добавить крупную кнопку "Начать создавать" между полем ввода промпта и примерами промптов (после строки 73, перед строкой 75)
- Кнопка будет:
  - Полная ширина на мобильном, авто-ширина на десктопе
  - Крупный размер текста (`text-lg md:text-xl`)
  - Большие внутренние отступы (`py-5 px-10 md:py-6 md:px-14`)
  - Стиль `btn-premium` с эффектом свечения (`glow-purple`)
  - Иконка `ArrowRight` справа
  - Скруглённые углы (`rounded-2xl`)
  - Анимация при наведении (уже есть в `btn-premium`)
- Обёрнута в `motion.div` с анимацией появления (задержка чуть больше, чем у поля ввода)
- При клике — переход на `/editor`
